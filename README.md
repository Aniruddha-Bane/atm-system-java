# ATM System Java

## Overview

A console-based ATM application built in Java using Object-Oriented Programming principles.

Features include user authentication, deposits, withdrawals, balance checking, transaction history, and file-based persistence.

## Features

* Account Number + PIN Login
* Multi-user Support
* Check Balance
* Deposit Money
* Withdraw Money
* Transaction History
* Persistent Storage using Files
* Audit Logging

## Technologies

* Java
* OOP
* File Handling
* Collections Framework

## Project Structure

model/
service/
util/
main/

## How To Run

1. Clone repository
2. Open in IntelliJ IDEA
3. Run Main.java
